# Barrierefrei Check — BFSG & WCAG accessibility check for WordPress

Honest accessibility auditing for WordPress, built for the German market — **BFSG**
(Barrierefreiheitsstärkungsgesetz) / EU Accessibility Act / **WCAG 2.2**. It scans the
**real rendered HTML** of your pages, reports prioritised issues with concrete fix
guidance, and generates a BFSG-compliant **Barrierefreiheitserklärung**. It is an honest
audit tool — **not an overlay** that masks problems.

## ▶ Live demo (no install)

**[Open the demo in WordPress Playground](https://playground.wordpress.net/?blueprint-url=https://raw.githubusercontent.com/kipphard/barrierefrei-check/main/demo/blueprint.json)**
— a full WordPress with the plugin installed and a sample audit already populated, running
entirely in your browser.

## Install

Download the latest [release](https://github.com/kipphard/barrierefrei-check/releases) zip
→ WordPress admin → *Plugins → Add New → Upload Plugin → Activate*. A new **Barrierefreiheit**
menu appears.

## Features (free)

- 12 automated WCAG 2.2 / BFSG checks (alt text, form labels, heading structure, landmarks,
  language attribute, link text, and more)
- Findings prioritised by severity, each with a plain-language fix and the exact WCAG criterion
- Barrierefreiheitserklärung (accessibility statement) generator
